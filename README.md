# Online Store – Spring Boot Application

Данное приложение представляет собой демонстрационный прототип интернет-магазина, реализованный на платформе Spring Boot. Проект включает функциональность каталога товаров, корзины, оформления заказов и административной панели для управления ассортиментом.

## Основные возможности

- Каталог товаров с фильтрацией по категориям и поиском по названию
- Функционал корзины с возможностью изменения количества и удаления товаров
- Оформление заказа и формирование итоговой квитанции
- Управление товарами и категориями через административную панель
- Регистрация и авторизация пользователей с разграничением ролей (ROLE_USER, ROLE_ADMIN)
- Формы входа и регистрации
- Интерфейс, реализованный с использованием Thymeleaf и Bootstrap

## Требования к среде

- Java версии 17 или выше
- Maven версии 3.6+
- MySQL версии 8.0+
- Предварительно созданная база данных 

## Настройка базы данных

Приложение подключается к существующей базе данных postgresql. Перед запуском необходимо создать таблицы:

- `users` — учетные записи пользователей
- `user_roles` — привязка ролей пользователей
- `categories` — категории товаров
- `products` — товары
- `orders` — заказы
- `order_items` — элементы заказа



## Конфигурация подключения

В файле `src/main/resources/application.properties` необходимо указать параметры подключения:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/onlinestore?useSSL=false&serverTimezone=UTC&allowPublicKeyRetrieval=true
spring.datasource.username=your_username
spring.datasource.password=your_password
